# mediasoup-server

Run the MedaiSoup Server:

```bash
npm install 

npm run start

npm run watch
```

Open [https://127.0.0.1:3000/broadcast/room1](https://127.0.0.1:3000/broadcast/room1) with your browser to see result.

# 기술 stack
- Node.Js
- Gstreamer
- MediaSoup
- WebRTC
- Socket
- HLSPlayer
